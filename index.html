CheaterJS

Installation Instructions:

  1. Have a HTML file, which imports your main javascript file as a module. Learn How
  2. Press the download button and drag cheaterJS into the folder you are using cheaterJS.
  3. Add This to the top of your main javascript file: import cheater from './cheater.js' (Replace ./cheater.js to the path to cheaterJS if nessesary.)

Usage Instructions:

If you successfully followed all above Instructions, any file imported with cheaterJS (step-3) would have access to cheater JS.
